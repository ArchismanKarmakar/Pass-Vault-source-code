Usage

Pass Vault is a password management program/system that works using Microsoft DotNET Framework 4.8. PassVault uses system generated hash and clock for random cryptography. The US certified AES or Advanced Encryption System as the best way of encrypting passwords. PassVault uses such AES techniques and special algos for encrypting and storing user passwords. PassVault's AI can switch b/w these 3 encrypion algos depending on the user password strength. It uses Windows DPAPI to store the encrypion keys securely in your OS. We will discuss about this in details later.

Password Criteria and Encryption

The best password format is "alphabets""ALPHABETS""NUMBER""SPECIAL_CHARS""AlPhAbEtS_&_NuMbErs" and this format can be rearranged into 5! new combinations.

Why disable internet connection while using??

This is not silly. It's serious. PassVault saves all user passwords in ACCDB in ultra encrypted form. But that doen't mean that a hacker cannot break the ACCDB file and get the passwords. Don't worry if such things happen then he will get those in encrypted formats & it's guratanted by AES founder that without the encryption code and DES code the encryption cannot be broken. So it's safe. But disabling internet adds an extra security. When PassVault is in use it will unlocks the ACCDB and retreives the data for user and now if your internet is on and a hacker seeks into your screen then what? or let's say someone breaks the connection and deletes all data then? So for extra security always use PassVault disconnecting your LAN port or WiFi.

Minimum System Requirements

  1. CPU: At least a x64 based CPU with atleast 2 cores 2 threads & 512 MHz operational speed with support of TPM 2.0.
  2. RAM: At least 512 MB of free System RAM must be available for smooth usage.
  3. GPU: At least 32 MB of Dedicated or Integrated GPU Memory must be free for a proper GUI experience.
  4. Storage: At least 1.5 GB of free storage is needed (installation on SSD: optional).
  5. OS: At least a NT based Windows 32-bit(x86) or 64-bit(x64) OS is needed i.e. Windows 2000, xp, 7, Windows 10, Windows 11 (all mentioned here are NT based only).
  6. Display: A display with resolution of at least 1280 x 720 is needed.
  7. Internet Connectivity: Not necessary with current version.
